### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DovydasTamasauskas/currency-exchange-be.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

Start the project:

```sh
node index.js
```
